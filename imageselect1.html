How to Use

  1. Configure the base URL above to match your GitHub repository
  2. Scroll through the gallery to find the file you need
  3. Click on any file card to copy the full URL to your clipboard
  4. A notification will confirm the copy action with the copied URL
  5. Use the search box to filter files by filename

Tip: The URL is dynamically generated using your base URL + filename. You can change the base URL anytime.
